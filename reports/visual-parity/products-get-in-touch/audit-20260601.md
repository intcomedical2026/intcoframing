# Products Get In Touch Visual Fix

Date: 2026-06-01

## Scope

- Product landing: `/products`
- Root product categories: `/mirror`, `/picture-frame`, `/art`, `/furniture`, `/memo-board`
- Global visible page ending: remove rendered `Frequently Asked Questions` and `Evidence and Claims` sections from all pages.

No deploy, commit, push, Vercel, Sanity, GTM, sitemap, robots, redirect, or LeadsCloud ID/configuration changes were made.

## Changes

- Removed the visible `SeoContentSections` render from `src/app/[[...slug]]/page.tsx`.
- Removed the source-mismatching intro paragraph above product Get In Touch forms.
- Styled only `.intco-leadscloud-main-form` so the real LeadsCloud form markup matches the source product form:
  - desktop two-column fields,
  - 483px by 80px desktop inputs,
  - 350px by 40px mobile inputs,
  - full-width message textarea,
  - centered 304px by 84px submit button.
